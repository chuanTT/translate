# Tích hợp api dịch

- đầu vào trong config/input.txt
- lang.txt list ngôn ngữ cần dịch

```bash
npm run trans
```

- đầu vào data.json --> input.txt

```bash
npm run toInput
```

# Chưa tích hợp api dịch

- đầu vào data.json --> vi.json

```bash
npm start
```

- đầu vào vi.json --> data.json

```bash
npm run sd
```
